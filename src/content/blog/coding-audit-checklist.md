## Conduct a practical coding audit

A coding audit should focus on documentation, code selection, and adherence to payer rules. Use a repeatable checklist to keep reviews consistent.

### Key audit steps

1. Confirm procedure documentation matches CPT and HCPCS codes.
2. Validate diagnosis codes support medical necessity.
3. Verify modifier use for separate services and bilateral procedures.
4. Review face-to-face encounter notes for E/M level support.

## Document exceptions clearly

When a claim is denied for coding reasons, document the appeal path, the right code bundle, and the corrective action for the next claim cycle.
