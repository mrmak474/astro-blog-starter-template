# MedBillingAndCoding

A medical billing and coding blog built with Astro 5, TypeScript, Tailwind CSS, and Cloudflare Pages.

## Features

- Astro 5 + TypeScript
- Tailwind CSS styling
- Cloudflare Pages deployment ready
- Homepage, Blog, Categories, Tags, Search
- Sitemap and RSS feed
- Structured schema metadata
- Professional medical billing monetization layout

## Quick start

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run locally:
   ```bash
   npm run dev
   ```
3. Build for production:
   ```bash
   npm run build
   ```

## Cloudflare Pages

Set the build command to `npm run build` and the output directory to `dist`.
